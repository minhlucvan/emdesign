# figma Taste Profile

**Brand fingerprint:** Figma's brand is a rigorous monochrome editorial frame (pure black on pure white, figmaSans variable type, pill-shaped CTAs) that repeatedly drops into oversized pastel color blocks — lime, lavender, cream, mint, pink, coral, and deep navy. The voice is confident, technical, and editorial with joyful interludes. The personality reads like a designer placing giant sticky notes on a clean desk: serious about craft, not afraid of color. The target audience is designers, product teams, and developers evaluating or implementing Figma's marketing design language.

**Design language:** Seven core principles govern the system. (1) Token binding — every value is a named token, never hardcoded. (2) Monochrome core with pastel accent blocks — black-and-white chrome makes color blocks feel intentional. (3) Pill is the only button shape — no square CTAs anywhere. (4) Weight-based hierarchy — body hierarchy comes from figmaSans variable weight (320, 330, 340, 480, 540, 700), never from opacity or mid-gray text. (5) Negative letter-spacing scales with display size — tight tracking on headlines, near-zero on body. (6) Color blocks as depth device — saturated background panels substitute for shadows and gradients. (7) Generous section rhythm — 96px vertical gap between major sections, white canvas separates every colored panel.

**Visual characteristics:** The system lives on six visual concepts. Monochrome canvas — pure white surfaces, pure black ink, black-and-white CTAs — establishes technical credibility. Oversized pastel color-block sections define the narrative rhythm of every long-form page. The sticky-note aesthetic means color blocks read as giant notes placed on a clean desk, not as decorative accents. Editorial typography uses tight line-heights on display sizes (1.00-1.10), generous on body (1.40-1.45), with fine weight increments from a single variable face. Pill-shaped CTAs are universal — every call-to-action is a rounded 50px pill. Mono taxonomy reserves figmaMono for uppercase eyebrows and captions only, never for body copy.

**Anti-patterns:** Avoid mid-gray text — hierarchy comes from weight, not opacity. Never add drop shadows to color-block sections; the color IS the depth device. Do not introduce new accent colors outside the documented block palette and accent magenta. Never combine more than one color block visible inside a single viewport — white canvas must separate them. Do not square off CTAs; square buttons read as a different brand. Never use figmaMono in body copy — it is a taxonomy tool. Do not replace the pricing-tab-selected black fill with a colored tab. Do not use primary black as a decorative accent — reserve it for genuine CTAs and selected states. Treat accent magenta as a single-shot color (one promo CTA per page, never two).
